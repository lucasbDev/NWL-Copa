# sales_api
## Descrição
- Este é o Back-end do projeto de Página de Bolão desenvolvido durante a Copa do Mundo de 2023

## Instalação

```bash
$ yarn 
```

## Rodando a aplicação

```bash
# development
$ yarn dev
```
